벡터는 위치 혹은 방향과 크기를 나타냄
두 벡터가 평행이동으로 완전히 겹쳐질 수 있다면 두 벡터는 같은 벡터임 (?)
특별한 경우는 제외하고 모든 벡터는 원점을 기준으로 해서 (x, y, z)만큼 평행이동 했다고 생각

벡터의 덧셈 = (x1 + x2, y1 + y2, z1 + z2)
벡터의 뺄셈 = (x1 - x2, y1 - y2, z1 - z2)
벡터의 곱셈 = A * (x, y, z) = (A * x, A * y, A * z) 
벡터의 크기 = sqrt(x^2 + y^2 + z^2), (3차원의 경우)
벡터의 크기를 바꾸거나 방향을 바꾸기 위해 곱하는 수를 스칼라 혹은 스케일이라고 함

벡터의 뺄셈은 점에 대해서 하는 것
점 A, B에 대해서 A - B는 B에서 A로 향하는 길, 두 점간의 관계를 알 수 있다.

소문자로 썼다면 방향벡터임, 그러나 방향 벡터나 점 벡터는 같다고 봄
두개의 방향벡터가 주어졌다면 점 3개가 주어진 것과 같음 (a, b, 원점)

모든 벡터를 단위벡터로 만든다면, 크기를 생각하지 않아도 되게 됨
절대값 벡터의 크기 분의 1로 나눠주면 단위 벡터가 됨
벡터의 정규화

초속 = x/초
1초 = 60프레임일때 1프레임에 x/60 만큼 이동  
어떤 벡터를 단위벡터로 만들고 초속 xm만큼 이동할 수 있다고 할때 단위벡터에 곱할수있다
B-A의 크기 = 두 점 사이의 거리

벡터의 외적은 또 다른 3차원 좌표를 만들어 낸다. A와 B에게 수직인 점
벡터의 외적은 3차원 벡터에서만 가능
법선벡터 : 방향을 나타내는 단위 벡터, 다각형의 법선 벡터는 다각형의 면에 수직인 벡터

벡터의 외적은 교환법칙이 성립되지 않음
